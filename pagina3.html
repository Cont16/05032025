TEMA DESCRIPCION
¿Qué es CSS? CSS (Cascading Style Sheets o Hojas de Estilo en Cascada) es un lenguaje de diseño utilizado para controlar la apariencia de las páginas web. Se usa junto con HTML para definir cómo se ven los elementos en un sitio web, incluyendo colores, fuentes, tamaños, espaciado y diseño.
TIPOS DE CSS
    Se puede incluir CSS de tres formas:
  1. CSS en línea Se aplica directamente en la etiqueta HTML usando el atributo style. Es útil para cambios rápidos, pero no recomendado para grandes proyectos porque hace el código difícil de mantener.
  2. CSS interno Se coloca dentro de una etiqueta style en la misma página HTML. Se usa cuando los estilos solo afectan a esa página.
  3. CSS externo Se escribe en un archivo .css separado y se enlaza en el HTML con la etiqueta link. Es el método más recomendado porque mantiene el código limpio y organizado.
Estructura de CSS Cada regla de CSS está compuesta por tres partes principales: 1️⃣ Selector → Indica a qué elementos HTML se aplicará el estilo. 2️⃣ Propiedad → Define qué aspecto del elemento se modificará (color, tamaño, margen, etc.). 3️⃣ Valor → Especifica el ajuste de la propiedad elegida.
VENTAJAS DE USAR CSS
  1. Separación de contenido y diseño Permite mantener el HTML limpio y estructurado, separando el estilo en archivos externos. 2️⃣ Facilita la reutilización Un solo archivo CSS puede ser usado en varias páginas, reduciendo la repetición de código. 3️⃣ Mejora la velocidad de carga Reduce el código HTML, permitiendo que la página se cargue más rápido. 4️⃣ Diseño responsive Permite adaptar el diseño a diferentes dispositivos (PC, tablet, móvil) con media queries. 5️⃣ Mayor facilidad de mantenimiento Al hacer cambios en un solo archivo CSS, todas las páginas que lo usan se actualizan automáticamente. 6️⃣ Efectos y animaciones CSS permite agregar transiciones, animaciones y efectos sin necesidad de JavaScript.
DESVENTAJAS DE USAR CSS Compatibilidad con navegadores Algunos estilos pueden verse diferentes en distintos navegadores y requieren ajustes específicos. 2️⃣ Curva de aprendizaje Aunque CSS es fácil de aprender al inicio, dominarlo completamente puede ser complicado. 3️⃣ Problemas con la especificidad Si hay muchas reglas CSS en un proyecto grande, puede ser difícil saber cuál tiene prioridad. 4️⃣ No es un lenguaje de programación CSS solo define estilos, por lo que no puede realizar lógica ni cálculos avanzados sin ayuda de JavaScript. 5️⃣ Puede volverse difícil de mantener Si no se organiza bien el código, con muchas clases y estilos repetidos, el CSS se vuelve desordenado..
